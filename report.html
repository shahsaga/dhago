Project Report

Shubham Adhikari my brother friend, launching an online Nepalese clothing brand. So he asked me to make him a website. For this project, I created a website for Dhago Clothing, a Nepalese clothing brand that aims to combine traditional designs with modern fashion. The purpose was to build a stylish, responsive website with at least four pages, using only HTML5 and CSS3.

Process & Planning

I began by outlining the requirements: the website would have a home page, a search button, a shop page containing products, a about page that would highlight the brand's history, and a contact page that would have a map and form. Prior to coding, I also made basic layout sketches and a site map. I utilised a dark blue colour scheme, Google Fonts for a contemporary appearance, and green to symbolise Nepalese culture through its relationship to nature.

Challenges

The biggest obstacle was that I was in my home country and couldn't physically attend class, so I missed the coding practice session when I could ask all of my questions in person, something I don't think was feasible online. Making the website responsive for desktop and mobile visitors is one of my other issues. For that, I also sought some assistance from chatgpt. This problem was resolved with the use of CSS grid and flexbox. Another difficulty I faced was making sure the gallery images loaded at the right sizes without causing the site to lag. However, your lesson, which included instructions on how to utilise GIMP, was helpful.

What I Learned

This project taught me how to use semantic tags, to organise HTML. Additionally, I worked on constructing a responsive navigation menu, adding hover effects, and creating layouts using CSS Grid and Flexbox. I looked at utilising the W3C validators for HTML and CSS to check my code.

Reflection

Designing the Shop page and playing around with the product cards was the most fun aspect. Setting up responsive graphics  was the most challenging aspects. Overall, I believe I met the assignment's objectives and developed my confidence in building websites from the ground up, even if I did get some assistance from chatgpt. I also made an effort to understand the reasons behind my decision to make our site responsive.

Time & Effort

In a week, I spent 3 days (1-2 hour each day) planning, coding, testing, and writing this report (total approx=50 hours). This included time spent searching for ideas, selecting images, and adjusting the CSS to make the site look consistent.
